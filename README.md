# Find-Yourself-in-Big-Data

Сканер персональных данных для корпоративных файловых хранилищ.  
Решение разработано в рамках хакатона **Samsung IT Академия 2026**.

---

## 🚀 Возможности

- **Рекурсивное сканирование** директорий с поддержкой более 10 форматов файлов.
- **Извлечение текста** из:
  - структурированных данных (`CSV`, `JSON`, `Parquet`)
  - офисных документов (`DOC`, `DOCX`, `RTF`, `XLS`, `XLSX`)
  - `PDF` (два парсера для повышения надёжности)
  - веб-страниц (`HTML`)
  - изображений (`TIF`, `JPEG`, `PNG`, `GIF`) — с опциональным OCR
  - видео (`MP4`) — покадровое распознавание документов и текста
  - простых текстовых файлов (`TXT`, `MD`)
- **Обнаружение персональных данных** согласно **152-ФЗ**:
  - ФИО, телефон, email, дата и место рождения, адрес
  - паспорт РФ, СНИЛС, ИНН, водительское удостоверение, MRZ
  - банковские карты (проверка алгоритмом Луна), счета, БИК, CVV
  - биометрические данные (ключевые слова)
  - специальные категории (здоровье, религия, политика, национальность)
- **Классификация уровня защищённости** (УЗ‑1 … УЗ‑4) по требованиям 152‑ФЗ.
- **Генерация отчёта** в форматах **CSV**, **JSON** или **Markdown**.
- **Многопроцессорная обработка** для высокой производительности.
- **Потоковое чтение** больших файлов (CSV, JSON, Parquet, Excel) — защита от переполнения памяти.
- **Предобработка изображений** (контраст, бинаризация, удаление шума) для повышения точности OCR.
- **Обработка повреждённых и неподдерживаемых файлов** без остановки сканирования.

🧭 Использование
Базовый запуск:

bash
python pd_scanner.py /путь/к/директории
Аргументы командной строки
Аргумент	Описание
directory	Обязательный. Путь к сканируемой папке.
-o, --output	Имя выходного файла отчёта (по умолчанию report.csv).
-f, --format	Формат отчёта: csv, json или md (по умолчанию csv).
--ocr	Включить распознавание текста в изображениях и видео.
-w, --workers	Число параллельных процессов (по умолчанию равно количеству ядер CPU).
-v, --verbose	Подробный лог выполнения.
Примеры
Сканирование без OCR, отчёт в JSON:

bash
python pd_scanner.py /data/dataset -f json -o report.json
Полное сканирование с OCR и 8 процессами:

bash
python pd_scanner.py /data/dataset --ocr -w 8 -f md -o report.md -v

🛠 Дополнительные утилиты
make_result.py
Конвертирует JSON-отчёт сканера в CSV-файл result.csv с колонками size,time,name (размер, время модификации, имя файла). Полезен для формирования списка файлов, в которых найдены ПДн.

bash
python make_result.py report.json result.csv
convert_report_csv.py
Аналогичная утилита, но работает с исходным CSV-отчётом сканера. Используется для обратной совместимости.

bash
python convert_report_csv.py report.csv result.csv

⚙️ Производительность и ограничения
Память: для CSV, JSON, Parquet, Excel и PDF установлены разумные лимиты (количество строк / символов). Это позволяет обрабатывать датасеты объёмом в несколько гигабайт без исчерпания RAM.

Скорость: многопроцессорность даёт линейный прирост на многоядерных системах. Обработка 3000+ файлов (~3 ГБ) на современном CPU с включённым OCR занимает от нескольких минут до десятков минут в зависимости от доли изображений и видео.

Точность: регулярные выражения дают некоторое количество ложных срабатываний (особенно для адресов и ФИО). Валидация по контрольным суммам (СНИЛС, ИНН, карты) существенно снижает число ошибок.
